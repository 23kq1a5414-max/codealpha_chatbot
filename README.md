# 🗨️ Python Chatbot

A lightweight, rule-based chatbot built in Python.  
This project demonstrates how to handle user input, maintain conversational context, and provide simple interactive responses.

---

## 📌 Overview
This chatbot is designed for beginners learning Python.  
It uses conditional logic to respond to user inputs and stores the user’s name for personalized interactions.

---

## ✨ Features
- Greets users and remembers their name
- Responds to basic commands:
  - `hello`
  - `my name is ...`
  - `how are you`
  - `bye`
  - `tell me a joke`
  - `help`
- Random joke generator
- Simple fallback for unrecognized input

---

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Libraries:** Standard Python libraries (`random`)
